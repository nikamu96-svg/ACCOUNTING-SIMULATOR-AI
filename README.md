# Accounting Simulator AI

Aplikasi simulasi akuntansi interaktif berbasis Streamlit dan Groq AI.

## Fitur
- Simulasi keputusan bisnis
- Laporan laba rugi real-time
- AI Accounting Coach

## Cara Menjalankan
1. Install dependency:
   pip install -r requirements.txt
2. Set API Key Groq:
   export GROQ_API_KEY="API_KEY_KAMU"
3. Jalankan:
   streamlit run app.py
